# 基于Flask的computer-science抽认（记忆）卡应用

在学习的过程中，我经常查看数小时的书籍or视频，并记录大量的笔记。可是一段时间后的我，依然会忘却其中大部分的东西。
所以我尝试翻阅笔记，并将需要加深记忆的东西制作成抽认卡（flashcard）。

你可以添加两种类型的抽认卡：概念、附代码的

而且，我会采用响应式去设计网页，使得在任何地方的我，都可以拿手机等移动设备回顾知识。

- CS概念知识

- Code
    - 数据结构
    - 算法
    - 疑难问题
    - 代码示例
    
    
项目使用了：

- Python3
- Flask
